Suppose \(\alpha,\beta\in\sym{n}\) are disjoint cycles and \(x\in[n]\text{.}\) First assume \(x\in\supp{\alpha}\text{.}\) Since \(\supp{\alpha}\cap\supp{\beta}=\emptyset\text{,}\) it must be that \(x\in\fix{\beta}\text{.}\) However by CheckpointÂ 2.1.26 we know \(\alpha(x)\in\supp{\alpha}\text{,}\) so likewise \(\alpha(x)\in\\fix{\beta}\text{.}\) But then
\begin{equation*} (\alpha\beta)(x) = \alpha(\beta(x)) = \alpha(x) = \beta(\alpha(x)) = (\beta\alpha(x)). \end{equation*}
By an analogous argument, if we instead assume \(x\in\supp{\beta}\) then \((\alpha\beta)(x)=(\beta\alpha)(x)\text{.}\)
