Let \(\alpha, \beta \in \Zp\) be given with \(\alpha\gt \beta\text{.}\) Define the following initial conditions:
\begin{align*} \amp\amp\amp\amp s_0 \amp= 1 \amp t_0 \amp= 0\\ a_1 \amp= \alpha \amp b_1 \amp= \beta \amp s_1 \amp= 0 \amp t_1 \amp= 1 \end{align*}
For each \(k \geq 1\text{,}\) do the following:
  1. Compute \(q_k, r_k\) by The Division Algorithm such that \(a_k = b_k\cdot q_k + r_k\text{.}\)
  2. If \(r_k\neq 0\text{,}\) then:
    1. Compute \(t_k = t_{k-2} - q_k\cdot t_{k-1}\text{.}\)
    2. Let \(a_{k+1} = b_k\) and \(b_{k+1} = r_k\text{.}\)
  3. On the other hand, when \(r_k = 0\text{,}\) then do the following:
    1. The value \(r_{k-1}\) is the last nonzero remainder, so \(\gcd(\alpha,\beta) = r_{k-1}\text{.}\)
    2. If \(r_{k-1} = 1\text{,}\) then \(\beta\cdot t_{k-1} = 1\pmod{\alpha}\text{.}\) Thus \(t_{k-1}\) is the reciprocal of \(\beta\) modulo \(\alpha\text{.}\)
