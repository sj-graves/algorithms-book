In the second iteration, we find
\begin{equation*} b = r_1\cdot q_2 + r_2 \implies r_2 = b-q_2\cdot r_1\text{,} \end{equation*}
but we can replace \(r_1\) with the result of the previous iteration, giving
\begin{align*} r_2 \amp = b-q_2 \cdot r_1\\ \amp = b-q_2\left(s_1\cdot a + t_1\cdot b\right)\\ \amp = \left(0-q_2s_1\right)\cdot a + \left(1-q_2 t_1\right)\cdot b\\ \amp = s_2\cdot a + t_2\cdot b \end{align*}
