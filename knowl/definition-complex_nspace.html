Definition 4.1.3. The vector space \(\CV{n}\).

Let \(n\in\Zp\text{,}\) and define \(\CV{n}\) to be the set
\begin{equation*} \CV{n} = \set{ \left.\cvec{z_0 \\ z_1 \\ \vdots \\ z_{n-1}} \right| z_0,z_1,\dotsc,z_{n-1}\in\Comps}\text{.} \end{equation*}
For any
\begin{equation*} \vv=\cvec{z_0\\z_1\\ \vdots\\ z_{n-1}}\in\CV{n} \end{equation*}
and \(k\in\set{0,1,\dotsc,n-1}\text{,}\) we define \(\entry{\vv}{k} = z_k\text{,}\) the kth component of \(\vv\), and we say that when \(\vv_1,\vv_2\in\CV{n}\text{,}\) \(\vv_1 = \vv_2\) precisely when \(\entry{\vv_1}{k} = \entry{\vv_2}{k}\) for every \({k\in\set{0,1,\dotsc,n-1}}\text{.}\) We further define the following operations on \(\CV{n}\text{:}\)
Addition
If \(\vv_1,\vv_2\in\CV{n}\text{,}\) then \(\vv_1+\vv_2\in\CV{n}\) is the vector satisfying \(\entry{\vv_1+\vv_2}{k} = \entry{\vv_1}{k}+\entry{\vv_2}{k}\) for each \({k\in\set{0,1,\dotsc,n-1}}\text{;}\) and
Scalar multiplication
If \(\vv\in\CV{n}\) and \(z\in\Comps\text{,}\) then \(z\cdot\vv = z\vv\in\CV{n}\) is the vector satisfying \(\entry{z\vv}{k} = z\entry{\vv}{k}\) for each \({k\in\set{0,1,\dotsc,n-1}}\text{.}\)
