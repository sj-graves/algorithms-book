Objectives 4.4.

With a good understanding of how to implement vectors in the AlgoVector class, moving to an implementation of matrices is a matter of extension rather than new definitions.
  1. Create a class AlgoMatrix of matrices in Python, including the following methods:
    1. In Activity 4.6, __init__ and __repr__
    2. In Activity 4.7, dims(self) which returns the row and column dimensions of self as a tuple, as well as __str__, __getitem__, and __eq__
    3. In Activity 4.8, __mul__ and __rmul__
    4. In Activity 4.9, __add__ and __radd__
  2. Add the following additional methods to the AlgoVector class:
    1. In Activity 4.10, column(self, col_index) which returns the column vector in the index col_index of self, as well as columns(self) which returns a tuple of all column vectors of self
    2. In Activity 4.11, conjugate(self) which returns the conjugate matrix of self
    3. In Activity 4.12, augment(self, right) which performs augmentation of the matrix self by an object right of appropriate type and dimensions.
    4. In Activity 4.13, transpose(self) which returns the transpose matrix of self, described below.
  3. Test your AlgoVector by making some computations in your aam_proj5.py file.
