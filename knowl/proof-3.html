Proof

The proof is actually an extension of Euclid's Algorithm. We will again apply the Division Algorithm, but at each step we will rewrite the equation to solve for the remainder. In the first iteration, we see the following:
\begin{equation*} a = b\cdot q_1 + r_1 \implies r_1 = a - q_1 b\text{.} \end{equation*}
Hence we define \(s_1 = 1\) and \(t_1 = -q_1\text{,}\) so that
\begin{equation*} r_1 = s_1\cdot a + t_1\cdot b\text{.} \end{equation*}
In the second iteration, we find
\begin{equation*} b = r_1\cdot q_2 + r_2 \implies r_2 = b-q_2\cdot r_1\text{,} \end{equation*}
but we can replace \(r_1\) with the result of the previous iteration, giving
\begin{align*} r_2 \amp = b-q_2 \cdot r_1\\ \amp = b-q_2\left(s_1\cdot a + t_1\cdot b\right)\\ \amp = \left(0-q_2s_1\right)\cdot a + \left(1-q_2 t_1\right)\cdot b\\ \amp = s_2\cdot a + t_2\cdot b \end{align*}
In order to see the pattern fully emerge, the third iteration consists of
\begin{equation*} r_1 = r_2\cdot q_3 + r_3 \implies r_3 = r_1 - q_3\cdot r_2\text{,} \end{equation*}
and we can again replace the \(r_1\) and \(r_2\) with the results of the previous two computations. Thus we obtain
\begin{align*} r_3 \amp = r_1 - q_3\cdot r_2\\ \amp = (s_1\cdot a + t_1\cdot b) - q_3(s_2\cdot a + t_2\cdot b)\\ \amp = (s_1-q_3s_2)\cdot a + (t_1-q_3 t_2)\cdot b\\ \amp = s_3\cdot a + t_3\cdot b\text{.} \end{align*}
We have established enough iterations to suggest recursive formulas
\begin{equation*} s_n = s_{n-2} + q_n s_{n-1}\text{ and } t_n = t_{n-2} + q_n t_{n-1} \end{equation*}
for \(n\geq 3\text{,}\) where
\begin{align*} s_1 \amp= 1, \amp s_2\amp = -q_2s_1,\amp t_1\amp= -q_1\amp t_2\amp = 1-q_2t_1\text{.} \end{align*}
But then there is \(k\in\Zp\) such that \(\gcd(a,b) = r_{k-1} = s_{k-1}a + t_{k-1}b\text{.}\)
