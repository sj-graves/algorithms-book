Definition 4.1.11. Inner product.

Define a function \(\hip{\cdot}{\cdot}:\CV{n}\times \CV{n}\to \Comps\text{.}\) The function is an inner product if and only if it satisfies the following properties:
Conjugate symmetry
\(\hip{\vv_1}{\vv_2} = \conj{\hip{\vv_2}{\vv_1}}\) for every \(\vv_1,\vv_2\in\CV{n}\text{;}\)
Linearity in the first argument
\(\hip{z_1\vv_1+z_2\vv_2}{\vv_3} = z_1\hip{\vv_1}{\vv_3} + z_2\hip{\vv_2}{\vv_3}\) for all \(\vv_1,\vv_2,\vv_3\in\CV{n}\) and \(z_1,z_2\in\Comps\text{;}\) and
Positive-definiteness
\(\hip{\vv}{\vv}\gt 0\) whenever \(\vv\neq \vec{0}\in\CV{n}.\)
A complex vector space equipped with an inner product is called an inner product space.
