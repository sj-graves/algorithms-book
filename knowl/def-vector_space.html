Definition 4.1.2. Vector space.

Let \(V\) be a set where addition in \(V\) is denoted by \(\vv_1+\vv_2\) when \(\vv_1,\vv_2\in V\) and where elements of \(V\) can be multiplied by complex numbers; that is we can write \(z \vv_1\) when \(z\in\Comps\) and \(\vv_1\in V\text{.}\) Moreover, we must have a well-defined notion of equality within \(V\text{.}\) Then \(V\) is a vector space over \(\Comps\) if and only if all of the following conditions are satisfied:
Additive Closure
If \(v_1,v_2\in V\) then \(\vv_1+\vv_2\in V\text{.}\)
Scalar Closure
If \(z\in\Comps\) and \(\vv\in V\) then \(z \vv\in \Comps\text{.}\)
Commutativity
If \(\vv_1,\vv_2\in V\) then \(\vv_1+\vv_2 = \vv_2+\vv_1\text{.}\)
Additive Associativity
If \(\vv_1,\vv_2,\vv_3\in V\) then \(\vv_1+(\vv_2+\vv_3) = (\vv_1+\vv_2)+\vv_3\text{.}\)
Additive Zero
There is a vector \(\vec{0}\in V\) such that \(\vv+\vec{0} = \vv\) for all \(\vv\in V\text{.}\)
Additive Inverses
For each \(\vv\in V\) there is a vector \(-\vv\in V\) such that \(\vv+(-\vv) = \vec{0}\text{.}\)
Scalar Multiplicative Associativity
If \(z_1,z_2\in\Comps\) and \(\vv\in V\) then \(z_1(z_2\vv) = (z_1z_2)\vv\text{.}\)
Distributivity across Vector Sums
If \(z\in\Comps\) and \(\vv_1,\vv_2\in V\) then \(z(\vv_1+\vv_2) = z\vv_1 + z\vv_2\text{.}\)
Distributivity across Scalar Sums
If \(z_1,z_2\in\Comps\) and \(\vv\in V\) then \((z_1+z_2)\vv = z_1\vv+z_2\vv\text{.}\)
Scalar One
If \(\vv\in V\text{,}\) then \(1\vv = \vv\text{.}\)
If \(V\) is a vector space over \(\Comps\) then \(\Comps\) is the field of scalars of \(V\text{,}\) the elements of \(V\) are vectors, and when referring to \(V\) we say that the elements of \(\Comps\) are scalars.
