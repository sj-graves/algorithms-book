Algorithm 2.2.18. Verifying that a sequence is one-line notation.

Suppose that for some fixed \(n\in\Zp\) a sequence \(a = (a_i:i\in [n])\) is given, and let \(s\) be a function which sorts sequences in ascending order.
  1. Let \(M = \max(a_i:i\in[n])\text{.}\)
  2. If \(s(a) = s([M])\text{,}\) then \(a\) is the one-line notation of a permutation of \([M]\text{.}\)
